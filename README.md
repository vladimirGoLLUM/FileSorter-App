# 📂 FileSorter App
**Автоматический сортировщик файлов на Python с современным GUI**

Приложение для наведения порядка в файловой системе. Сортирует тысячи файлов по папкам (Фото, Документы, Видео, Архивы) за секунды.

> 🛠 Разработано с использованием AI-подхода и архитектуры CustomTkinter.

---

## ✨ Ключевые особенности

- 🎨 **Современный UI:** Тёмная тема, плавные элементы (CustomTkinter).
- ⚡ **Высокая скорость:** Многопоточность (Threading) — интерфейс не зависает при обработке 10,000+ файлов.
- 🔄 **Гибкие настройки:** Режимы «Переместить» или «Скопировать».
- 📊 **Умная логика:**
  - Рекурсивное сканирование (учитывает подпапки).
  - Защита от перезаписи (авто-переименование при конфликте имён).
  - Игнорирование системных файлов.
- 📝 **Отчетность:** Детальный лог операций + выгрузка отчёта в `.txt`.
- 🛡️ **Безопасность:** Предупреждение при попытке сортировки системных папок.

## 🚀 Установка и запуск

Вам потребуется **Python 3.11+**.

### Из исходников

```bash
# Клонируйте репозиторий
git clone https://github.com/pycraft-dev/FileSorter-App.git
cd FileSorter-App

# Создайте виртуальное окружение
python -m venv .venv
.venv\Scripts\activate  # Windows
# source .venv/bin/activate  # macOS/Linux

# Установите зависимости
pip install -r requirements.txt

# Запустите приложение
python main.py
```

### Сборка в EXE (Windows)

```bash
# Установите PyInstaller
pip install pyinstaller

# Соберите исполняемый файл
pyinstaller --onefile --windowed main.py

# Готовый файл будет в папке dist/
```

Или просто запустите `build.bat` в корне проекта.

## 🏗 Архитектура проекта

Проект создан с соблюдением лучших практик Python-разработки:

```
FileSorter-App/
├── main.py                 # Точка входа, инициализация GUI
├── src/
│   ├── core/
│   │   ├── file_sorter.py  # Логика сортировки (класс FileSorter)
│   │   ├── file_types.py   # Словари расширений по категориям
│   │   └── logger.py       # Настройка логирования в UI
│   ├── ui/
│   │   ├── main_window.py  # Класс MainWindow (CTk)
│   │   ├── components.py   # Кастомные виджеты
│   │   └── styles.py       # Цвета, отступы, константы
│   └── utils/
│       └── helpers.py      # Вспомогательные функции
├── requirements.txt
├── README.md
└── build.bat               # Скрипт сборки в EXE
```

- ✅ **Type Hinting:** Полная аннотация типов для надёжности.
- ✅ **Модульность:** Чёткое разделение логики, UI и утилит.
- ✅ **Безопасность:** Обработка ошибок без падения приложения.
- ✅ **Кириллица:** Корректная работа с путями на русском языке.
